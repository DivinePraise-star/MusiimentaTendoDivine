# Recess Data Science Assignments

This repository contains Python assignments for the Recess Data Science program, completed by **Musiimenta Tendo Divine** (24/U/07405/PS). Each script is designed to solve a specific problem and demonstrate fundamental Python concepts.

---

## Assignments

Here is a summary of the assignments included in this repository:

### 📄 Assignment 1: Bill Split Calculator
A command-line program that calculates how to split a bill among a number of people. It includes functionality for adding a tip (pre-defined or custom) and provides a formatted receipt of the final calculations.

- **File:** `Assignment1.py`
- **Concepts:** User input, type casting, mathematical operations, and f-strings for formatted output.

### 🛒 Assignment 2: E-commerce Platform Simulation
A simulation of a simple e-commerce platform with a role-based login system (Admin, Customer, Cashier). The program calculates the final price of a product, applying discounts and taxes based on different conditions.

- **File:** `Assignment2.py`
- **Concepts:** Dictionaries, functions, conditional logic (`if/elif/else`), and user-defined functions.

### ⚽ Assignment 3: World Cup Winner Guessing Game
A simple guessing game where the user tries to guess the winner of the 2026 World Cup. This assignment demonstrates the use of loop control statements.

- **File:** `Assignment3.py`
- **Concepts:** `while` loops, and loop control statements (`break`, `continue`, `pass`).

## 🧮 Assignment 4: GUI Calculator
A graphical user interface (GUI) calculator built with Python's tkinter library. It features a modern, dark-themed design with interactive buttons for performing standard arithmetic operations.

- **File:** Assignment4.py
- **Concepts:** Functions, GUI programming with tkinter, event handling, and layout management.
